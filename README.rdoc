== README

This hackathon project uses a custom Rack MiniProfiler to forward profiling data to Splunk Cloud.

* Relevant configuration -- https://github.com/hyfather/myblog/blob/master/config/initializers/mini_profiler.rb
* Forked Rack Miniprofiler -- https://github.com/hyfather/rack-mini-profiler

Installation Instructions

* _Recommended_ Install RVM and then Ruby 2.x (http://rvm.io/)
* git clone https://github.com/hyfather/myblog.git
* cd myblog
* bundle install
* bundle exec rails server
